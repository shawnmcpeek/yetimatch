# AdMob iOS setup

The iOS app uses CocoaPods for the Google Mobile Ads SDK.

1. In the `iosApp` directory run:
   ```bash
   pod install
   ```
2. Open **iosApp.xcworkspace** in Xcode (not the .xcodeproj).
3. Build and run the app.

Ad unit IDs and App ID are in the PDF / manifest; banner is shown on the result screen.
